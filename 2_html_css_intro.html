HTML: Definition

 

What is HTML?

HTML (HyperText Markup Language) is a markup language that uses a system of tags to dictate how the content should be displayed within a web browser.

CSS: Definition

What is CSS?

CSS (Cascading Style Sheets) is a computer language that was created to supplement HTML. CSS styles can be applied to an HTML page to position text and images and dramatically change the appearance of things like text formatting, link effects, etc. They work hand-in-hand to create a professional web site.

Working with a Code Editor

Step #1: Open Atom

Once you have the Atom software installed, open the software.

Step #2: Download Web Files

We are not building an entire web page from scratch, so start by downloading this .zip file,

Once you have the .zip file download, unzip the files. You should have a folder called "one-night" with all the web files you need within it. It can help at this point to move this folder onto your computer's desktop.

Step #3: Drag in the Project Folder

When working with web pages within code editors, you aren't just working on a single page, but multiple pages—such as each .html page and the .css page. Because of this, code editors tend to have a navigation along the left-hand side of the app that clearly lists all of the files and folders associated with a web project.

With Atom open, drag the "one-night" folder over the left-hand side of the application.

Step #4: Find the HTML page

Once you release the folder, you'll see it populated as a navigation along the left-hand side of the app. Feel free to click onto the webpage.html to see the source code of that document.

Note: You may not see the .DS_Store file when you open this project, but if you are on the mac, there is one there. The .DS_Store is typically a hidden file that is found on the Apple OS X operating system. The .DS_Store is a file that stores unique data about the files within the folder, such as the position of the icons and the background images to the folder. The name is just an abbreviation of Desktop Services Store. Weither you see it or not, feel free to ignore it.

Step #5: Take a Moment to Review Files

Seeing all this code and the crazy colors can be a little overwhelming at first—don't stress, you'll soon find our how useful this is and easy you'll start writing and editing code without worring about the details.

Take a moment to review the following files:

  • webpage.html
  • style.css
  • images folder
